WELCOME MIKROKERS

Mengenal Costum Chain Pada Mikrotik * Forward – Digunakan untuk memproses trafik paket data yang hanya melewati router. * Input – Digunakan untuk memproses trafik paket data yang masuk ke dalam router melalui interface yang ada di router. * Output – Digunakan untuk memproses trafik paket data yang keluar dari router. * Prerouting – Digunakan untuk memproses trafik paket data yang masuk kedalam router dan melewati router. * Postrouting – Digunakan untuk memproses trafik paket data yang akan keluar dari router. * Action Jump – Berfungsi untuk melompat ke chain lain yang telah didefiniskan pada paramater jump-target. * Action Return – Berfungsi untuk memproses kembali aturan yang kita buat.
