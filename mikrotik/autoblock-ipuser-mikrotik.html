MIKROTIK SCRIPT : Auto Block Ip User Profile Pada Jam Tertentu

IP->Firewal->Filter Rules


# Paste script to terminal /ip firewall filter add action=jump chain=forward comment="drop jam tertentu Copyright Script by DNEL" jump-target=dnel-disable-time src-address-list=Ip-Limit time=10h30m-12h,sun,mon,tue,wed,thu,fri,sat add action=drop chain=dnel-disable-time dst-port=80,443 protocol=udp src-address-list=Ip-Limit add action=drop chain=dnel-disable-time dst-port=80,443 protocol=tcp src-address-list=Ip-Limit


IP->Hotspot->User Profile


# Paste script to terminal /ip hotspot user profile add address-list=Ip-Limit name=UserLimit rate-limit="512k/2M 0/0 0/0 0/0 8 56k/128k"


IP->Hotspot->User


# Paste script to terminal /ip hotspot user add name=dnel password=dnel profile=UserLimit
